# Getting Started

### npm
```
npm install @efelle/proton --save
```

### Yarn
```
yarn add @efelle/proton
```

### Module System
When using a module system, you must explicitly install Proton via `Vue.use()`:

```javascript
import Vue from 'vue'
import Proton from '@efelle/proton'

Vue.use(Proton)
```
